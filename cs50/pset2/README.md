# CS50 SQL PSet 2 Workspace

CS50 does not provide starter ZIP files for Problem Set 2. The official instructions expect you to create these folders yourself.

Ready-to-edit folders:

- `cs50/pset2/atl/`
- `cs50/pset2/connect/`
- `cs50/pset2/donuts/`

Each folder already contains a blank `schema.sql` placeholder so you can start writing immediately.
